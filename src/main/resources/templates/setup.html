Setup TeamTactics

Step 1: Install TeamTactics client software

Download and install the client software package on your iRacing computer.

It's recommended to install the software just for your own user - not for all users on your computer. 


The next step - configuration - is the last step in the installation procedure.
So do not finish your installation now!

Step 2: Configure client software

Go to your user profile and obtain your TeamTactics client access token.
Configure it together with your iRacing ID:

[DEFAULT]

[global]
# Each team member has to configure its own iRacing ID here like
# iracingId = 123456
iracingId = Yor iRacing ID here

# Proxy configuration. The given URL will be used as Proxy on both http and
# https protocol
;proxy =

# Generates additional debug output. All MQTT and serial communication will be
# logged. Comment out or set to yes/True to enable
;debug = yes
logfile = irtactics.log
# Uncomment to start the application using a data dump file from irsdk for
# testing/development purposes. The dump file can be created by issuing the
# command 'irsdk --dump data.dmp'
;simulate = data/monzasunset.dump

[connect]
# Fill in the client access token from your TeamTactics profile at
# https://iracing-team-tactics.appspot.com/profile
clientAccessToken = Your client access token here

# URL where the client send its messages to - usually no subject to change
postUrl      = https://iracing-team-tactics.appspot.com/clientmessage

You may now complete your software setup.

Step 3: Go team racing

Start the client software (There should be an Icon on your desktop) and go racing.
Follow your team members data on the TeamTactics Server.
